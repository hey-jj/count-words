# Changelog

## [0.2.0] - 2026-07-07

### Changed
- Latin text joined with multiplication or division signs now splits into separate words. (#14)
- Latin text joined to Malayalam digits now splits around the digit. (#15)

### Performance
- Counting input with many default punctuation marks is faster. (#17)
- Counting large input uses less temporary memory during token splitting. (#18)

### Documentation
- The README now explains that punctuation replacement runs before General Punctuation and Halfwidth and Fullwidth Forms are stripped. (#16)
